# Personal Academic Portfolio Website

## Overview

This project is a personal academic portfolio website created for the CS 463/563 Intro to Web Development Final Project at Portland State University.

The website highlights my academic background, research interests, professional experience, technical skills, publications, and selected projects. It was designed to provide a professional online presence while demonstrating the web development concepts covered throughout the course.

### Projects Section

#### Poker Chips Helper

A web application that assists users with poker chip calculations and distribution during games.

#### Rat Model Optimization

A computational research project that applies Particle Swarm Optimization (PSO) techniques to improve the performance of a neuromechanical rat locomotion model.

Each project includes a description and a link to its GitHub repository.

## Accessibility Considerations

The website was designed with accessibility in mind by:

* Using semantic HTML elements
* Including labels for form fields
* Providing alternative text for images
* Maintaining logical heading structure
* Creating responsive layouts
* Testing with accessibility auditing tools
